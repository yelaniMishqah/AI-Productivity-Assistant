# CareerBoost AI

## Project Overview

CareerBoost AI is an AI-powered career development platform designed to help students, graduates, and job seekers improve their employability and career readiness. The application leverages Artificial Intelligence to provide personalized career guidance, resume analysis, interview preparation, and career planning.

The goal of CareerBoost AI is to simplify the career development process by offering intelligent recommendations and actionable insights that help users make informed career decisions.

---

## Features

### Resume Analyzer

* Analyze uploaded or pasted resumes
* Generate a resume score
* Identify strengths and weaknesses
* Provide ATS (Applicant Tracking System) optimization suggestions
* Recommend improvements for better job application success

### AI Interview Coach

* Generate interview questions based on job role and industry
* Provide sample answers
* Offer interview preparation guidance
* Help users build confidence before interviews

### Career Planner

* Create personalized career roadmaps
* Recommend skills and certifications
* Define career goals and milestones
* Suggest learning paths for career growth

### AI Career Assistant Chatbot

* Answer career-related questions
* Provide job search guidance
* Recommend learning resources
* Assist with professional development planning

## Installation and Setup Instructions

### Prerequisites

Before running the project, ensure you have:

* Node.js installed
* Git installed
* OpenAI API key

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/careerboost-ai.git
```

2. Navigate to the project directory:

```bash
cd careerboost-ai
```

3. Install dependencies:

```bash
npm install
```

4. Create a `.env` file and add your API key:

```env
OPENAI_API_KEY=your_api_key_here
```

---

## How to Run the Application

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to:

```text
http://localhost:3000
```

The application should now be running locally.
